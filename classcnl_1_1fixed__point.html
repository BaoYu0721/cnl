Detailed Description

template<typename Rep, int Exponent, int Radix>
class cnl::fixed_point< Rep, Exponent, Radix >

literal real number approximation that uses fixed-point arithmetic

Template Parameters
Repthe underlying type used to represent the value; defaults to int
Exponentthe exponent used to scale the integer value; defaults to 0
Radixthe base used to scale the integer value; defaults to 2

Uses an integer to approximate a real number. Scales the integer by pow(Radix, Exponent) to produce the scaled number. By default, represents an int with no fractional digits using binary arithmetic.

Note
Currently, only binary support and partial decimal support is provided.
Examples

To define a fixed-point value 1 byte in size with a sign bit, 4 integer bits and 3 fractional bits:

constexpr auto n = fixed_point<std::int8_t, -3>{-2.75};
static_assert(n==-2.75, "fixed-point type was unable to store the value");
